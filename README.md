# NCKU_Web_Ajax
A simple example for NCKU_Web_Design course
## Step0: Set up
  * Install express and jquery by ```npm```
  
    ```
    $ npm i
    ```
    or
    
    ```
    $ npm init -y
    $ npm i express jquery --save
    ```
### Step2: Execute ser.js
  * Execute server at ```./```
  
    ```
    $ node ser.js
    ```
  * View result [HERE!](http://luffy.ee.ncku.edu.tw:5358/test.html)
